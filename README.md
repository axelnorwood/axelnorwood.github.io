# axelnorwood.github.io
🌟 Overview
This project is a lightweight, static website designed to support students, parents, and teachers with clear tools for literacy development. Each page focuses on a different aspect of reading or writing, giving families practical guidance they can use at home or in school.

The site is intentionally simple: fast to load, easy to navigate, and accessible on any device.

🗂️ Pages Included
Home Page (index.html)
A welcoming starting point that links to all learning tools.

Reading Atlas (reading-atlas.html)
A map of reading skills and behaviors to help families understand where a student is and what comes next.

Reading Checklist (reading-checklist.html)
A simple checklist students can use to track habits, strategies, and progress.

Writing Continuum (writing-continuum.html)
A developmental progression that shows how writing skills grow over time.

🎯 Purpose
The goal of this project is to make literacy support clear, friendly, and actionable.
Parents often want to help but don’t know where to start. Students benefit from seeing their growth. Teachers appreciate tools that reinforce classroom learning.

This site brings all of that together in one place.

🛠️ Tech Stack
HTML5 — simple, portable, and easy to maintain

CSS — minimal styling for clarity and readability

GitHub Pages — hosting via the included CNAME file

No frameworks, no dependencies — just clean, accessible content.

🚀 Getting Started
To run the site locally:

bash
git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>
open index.html
To deploy, enable GitHub Pages in your repo settings and ensure the CNAME file matches your custom domain (if using one).
